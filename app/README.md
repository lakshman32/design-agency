# Design Agency Website

A modern responsive website built using Next.js and Tailwind CSS.

## Live Demo
https://your-vercel-link.vercel.app

## Tech Stack
- Next.js
- React
- Tailwind CSS

## Features
- Responsive Navbar
- Hero Section with CTA
- Services Section
- Portfolio Grid
- Contact Form UI

## Setup Instructions

1. Clone repo:
git clone https://github.com/your-username/design-agency.git

2. Install dependencies:
npm install

3. Run project:
npm run dev

4. Open:
http://localhost:3000

## Notes
- Frontend only project
- Contact form is UI only
- Images stored in /public folder
